# JARVIS - Virtual Assistant  
JARVIS is a simple virtual assistant that can respond to voice commands, perform internet searches, and open commonly used websites like Google, YouTube, and Facebook.

---

## 💡 **Features**
✅ Voice command recognition  
✅ Responds with text-to-speech  
✅ Opens Google, YouTube, and Facebook  
✅ Answers general questions using Google search  
✅ Retrieves current time and date  
✅ Opens Wikipedia for specific information  

---

## 🚀 **How to Run**
 Clone the repository:  

git clone https://github.com/kmprasannakumar/Jarvis-AI-Assistant.git
Open the project folder:
cd Jarvis-AI-Assistant
Open index.html in your browser.

🎯 Commands
Command	Action
"Open Google"	Opens Google
"Open YouTube"	Opens YouTube
"Open Facebook"	Opens Facebook
"What is the time?"	Tells the current time
"What is the date?"	Tells the current date
"Search for [query]"	Searches the query on Google
"Wikipedia [topic]"	Opens Wikipedia for the topic
